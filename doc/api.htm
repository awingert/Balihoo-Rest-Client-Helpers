Customer List Management API Specification

HTTP Headers

Header Name Description Required
X-Auth-Token The API key supplied to you by Balihoo Yes

HTTP GET Service

Parameters Resulting Action Example URI
None All contacts are returned http://clm.balihoo.com/contact
Contact ID Only the requested contact is returned, if found. http://clm.balihoo.com/contact/4da4ca854fae881d3b0000a6
"status" and batch ID The status of the given batch is returned. Returns "completed" if the batch is not found to be running. http://clm.balihoo.com/contact/status/4da4ca854fae881d3b0000a6

HTTP DELETE Service

Parameters Resulting Action Example URI
Contact ID Deletes the contact matching the given ID, if found. http://clm.balihoo.com/contact/4da4ca854fae881d3b0000a6

HTTP POST Service

Parameters Resulting Action
command=load
contacts=[JSON Encoded Array of Contacts] 		Queues the given contacts for loading into your contact database. A Batch ID is returned that can be used to query the status of the loading process.
command=count
[optional]query=[JSON Encoded query]
[optional]view=[Email|DirectMail|MainPhone|MobilePhone] 		Returns a count of contacts matching the given query and distinct to the given view.
command=count
[optional]query=[JSON Encoded query]
[optional]view=[Email|DirectMail|MainPhone|MobilePhone] 		Returns a JSON encoded array of the contacts matching the given query and distinct to the given view. If two or more contacts are considered equal in the current view, the contact last updated will be returned.
